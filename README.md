# 🍪 Choco Bytes — «Солодка Майстерня»

## Вебзастосунок «Солодка Майстерня» — це односторінковий сайт для перегляду асортименту десертів, фільтрації за категоріями, перегляду деталей десерту, відгуків та оформлення замовлення через модальне вікно.

## ✏️ Про проєкт

**Задача, яку вирішує проєкт:**

- показує каталог десертів з бекенду;
- дає можливість фільтрувати/шукати десерти;
- відкриває модальне вікно з деталями десерту;
- відкриває модальне вікно оформлення замовлення та відправляє дані на бекенд;
- забезпечує адаптивність під брейкпойнти: **375 / 768 / 1440** (від 320 —
  гумово).

---

## 🛠 Технології

- **HTML5**, **CSS3**
- **JavaScript**
- **Vite** (збірка, dev-server)
- **Axios** (HTTP-запити до бекенду)
- **SweetAlert2** (нотифікації/підтвердження)
- **Choices.js** (select на tablet/mobile)
- **Swiper** (слайдер у секції відгуків / або інших секціях)
- **raty-js** (відображення рейтингу зірочками)

---

## 🖥️ Бекенд / API

Проєкт працює з бекендом через REST API.

- Документація API: `https://deserts-store.b.goit.study/api-docs/`

# 🍪 Choco Bytes — “Sweet Workshop”

“Sweet Workshop” is a single-page web application for browsing a dessert
assortment, filtering by categories, viewing dessert details and reviews, and
placing an order via a modal window.

---

## ✏️ About the Project

**What the project does:**

- displays a dessert catalog fetched from the backend;
- allows filtering/searching desserts;
- opens a modal window with dessert details;
- opens an order modal and sends order data to the backend;
- provides responsive layout for breakpoints: **375 / 768 / 1440** (fluid from
  320px).

---

## 🛠 Technologies

- **HTML5**, **CSS3**
- **JavaScript**
- **Vite** (build tool, dev server)
- **Axios** (HTTP requests to the backend)
- **SweetAlert2** (notifications / confirmations)
- **Choices.js** (select on tablet/mobile)
- **Swiper** (slider in the reviews section and/or other sections)
- **raty-js** (star rating display)

---

## 🖥️ Backend / API

The project works with a backend via a REST API.

- API documentation: `https://deserts-store.b.goit.study/api-docs/`
